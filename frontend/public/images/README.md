# AI 模型图标文件

请将以下图标文件放置在此目录下（`/public/images/`）：

## 必需的文件

1. **gemini-icon.png** - Gemini 图标（或 gemini.png, Gemini.png）
2. **chatgpt-icon.png** - ChatGPT 图标（或 chatgpt.png, ChatGPT.png）
3. **doubao-icon.png** - 豆包图标（或 doubao.png, 豆包.png, 豆包-icon.png）
4. **qianwen-icon.png** - 千问图标（或 qianwen.png, 千问.png, 千问-icon.png, qwen-icon.png）

## 文件格式要求

- **推荐格式**：PNG（支持透明背景）
- **推荐尺寸**：至少 64x64 像素，建议 128x128 或更大
- **文件大小**：建议每个文件不超过 500KB

## 文件命名说明

组件支持以下文件名格式（按优先级）：

### Gemini
- `gemini-icon.png` （首选）
- `gemini.png`
- `Gemini.png`

### ChatGPT
- `chatgpt-icon.png` （首选）
- `chatgpt.png`
- `ChatGPT.png`

### 豆包
- `doubao-icon.png` （首选）
- `doubao.png`
- `豆包.png`
- `豆包-icon.png`

### 千问
- `qianwen-icon.png` （首选）
- `qianwen.png`
- `千问.png`
- `千问-icon.png`
- `qwen-icon.png`

## 注意事项

- 如果图标文件不存在，组件会自动显示 emoji 备用图标
- 确保文件路径正确：文件应直接放在 `/public/images/` 目录下
- 在浏览器中访问路径为：`http://localhost:端口/images/文件名.png`
- 如果图标仍然不显示，请检查浏览器控制台的错误信息
